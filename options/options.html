Info

Standard: Default Vivaldi theme.

Custom: Create your own custom theme. Click edit and start picking colors. Press Activate to save to storage and load. You can import and export themes.

User CSS: Click edit to input your own custom css code.

:

Auto-hide header on scroll: Hides the header when you scroll down, shows it when you scroll up.

Bookmarks in navigation: Displays a link to the bookmarks page in the main navigation, for easy access.

Compact header: Sets the header to its original height and removes redundant links.

Custom notification icons: Displays additional font icons in the header's notification dropdown, to quickly distinguish one type of notification from another.

Hide tooltips: Hides header and navigation bar tooltips.

Hide unread count: Hides the unread count icon in the navigation bar.

Last edit timestamp: Displays the time of last edit on bottom of edited posts. Doesn't work well with infinite scroll! Please enable pagination of topics and posts on the Vivaldi Forum settings page.

Show User IDs: Shows UIDs when hovering the username at top of a post, and when hovering a user's avatar on their profile.

Signature mod: Hides all signatures and replaces each with a button. When clicked the signature of individual posts can be shown. Doesn't work well with infinite scroll! Please enable pagination of topics and posts on the Vivaldi Forum settings page.

Square avatars: Square avatars everywhere!

Default modifications: Including, but not limited to: Bookmarking fix, Voting indicators, Copy all code button, more space for expanded chat input and improved font icons. Additionally links to the options page and the hidden forum profile edit page are available from the avatar dropdown menu and links to the unofficial Discord chat and the store are placed in the footer.
